# English Version

## Overview

Smart tools to convert Microsoft Word footnotes and endnotes to plain text between `&&` markers for AI proofreading, then restore them to their original format after proofreading.

## Quick Usage

### Method 1: VBA Macros (Easiest)
1. Open your Word document
2. Press `Alt + F11` to open VBA editor
3. Insert the macro code
4. Run the macro
5. Proofread in AI
6. Run restore macro

### Method 2: Python Scripts
```bash
# Extract
python extract_footnotes.py input.docx

# Restore
python restore_footnotes.py output.docx
```
